# Library Management System

A simple Java-based library management system for managing books, members, and borrowing operations.

## Features

- Add, list, and search books
- Add and list members
- Borrow and return books
- Track borrowed books

## Technologies Used

- Java (JDK 11 or higher)
- Standard Java Collections Framework

## How to Run

1. Make sure you have Java installed
2. Compile the program: `javac LibraryManagementSystem.java`
3. Run the program: `java LibraryManagementSystem`

## Project Structure

- `LibraryManagementSystem.java`: Main class with the application logic
- `Book.java`: Book entity class
- `Member.java`: Member entity class
- `BorrowRecord.java`: Borrow record entity class
